Hackers of Sweden

To become a member please go to repository and push Pull Requests with a new file to directory members. File name should be [your github login].html To be able to be a member you need to explain how do you connect to Sweden for exemple born in sweden, working in sweden, working for swedish company, ect. Don't worry if you are not connected to Sweden you can put your pull request to mapp supporter.
